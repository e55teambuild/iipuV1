     Программа  разработана  при помощи Qt Creator  средствами QtQuick.
     При создании  программы  был разработан интерфейс, который позволяет реализовать работу  со структурными компонентами проекта, такими как «работа с периферийными устройствами ПК», «получение информации из системных директорий:devices,cpuinfo,meminfo», « демонстрация модуля работы с клавиатурой» , «адаптер асинхронного последовательного интерфейса RS-232».
     Возможность страничной навигации внутри окна приложения разработана для удобства обращения пользователя со структурными разделами программы.  В программе предусмотрена возможность замены фона окна приложения, это дает возможность пользователю самостоятельно выбирать для себя комфортный дизайн при работе с программой.
 	Один из компонентов программы – это получение информации о подключенных устройствах. Реализация данного элемента выполнено при помощи команды dmesg.
     Следующий компонент проекта – получение информации из системных директорий, таких как devices,cpuinfo,meminfo. Devices отображает различные блочные и символьные устройства, подключенные к системе, кроме тех, для которых не загружены модули ядра. Cpuinfo хранит подробную информацию о процессоре. Meminfo содержит отображение всей доступной информации об оперативной памяти и памяти подкачки.
 	Реализация модуля работы с клавиатурой – третий компонент программы. Предназначен для наглядной демонстрации обработки нажатий на клавиатуре.
     Последним компонентом программы является разработанный алгоритм  взаимодействия с адаптером последовательного интерфейса.
